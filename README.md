# EcoTrack 🌿 — Your AI Sustainability Partner

EcoTrack is an intelligent waste classification system that gamifies the experience of recycling to help save the planet.

![EcoTrack App Concept](https://images.unsplash.com/photo-1532996122724-e3c354a0b15b?auto=format&fit=crop&q=80&w=1200)

---

## 🌎 Concept: 5W1H

### **Why? (The Purpose)**
Landfills are overflowing due to improper waste sorting. Proper disposal can reduce landfill waste by up to **60%**. EcoTrack aims to bridge the gap between "wanting to recycle" and "knowing how to recycle."

### **What? (The Solution)**
A full-stack web application that uses **AI Computer Vision** to instantly identify materials. It features a point-based system, global leaderboards, and a comprehensive sorting guide to make sustainability fun and addictive.

### **Who? (The Audience)**
Eco-conscious individuals, households, and environmentally responsible communities who want to track their personal impact and compete with others to see who can save the most CO2.

### **Where? (The Accessibility)**
Accessible anywhere via a mobile or desktop web browser. Whether you are at a cafe, the office, or at home, EcoTrack helps you sort your trash on the spot.

### **When? (The Usage)**
Every time you have a piece of waste! With **Daily Streaks** and **Leagues**, the app encourages consistent, daily habits that lead to long-term environmental change.

### **How? (The Process)**
1. **Scan**: Point your camera at an item.
2. **Identify**: AI classifies it as Recyclable, Organic, General, or Hazardous.
3. **Earn**: Gain points and climb the "Silver" and "Gold" leagues.
4. **Track**: Monitor your CO2 saved and impact history on your personal dashboard.

---

## ✨ Key Features

- 📸 **AI-Powered Scanning**: Real-time material identification using Gemini AI.
- 🏆 **Competitive Leagues**: Compete with friends and the global community.
- 📘 **Smart Sorting Guide**: A visual encyclopedia for waste disposal.
- 🌍 **Multi-language Support**: Seamlessly toggle between **English** and **Thai**.
- 🔥 **Streak System**: Gamified motivation to keep your recycling habits consistent.
- 🔒 **Firebase Integration**: Secure user profiles and real-time activity synchronization.

## 🛠️ Tech Stack

- **Frontend**: React 18+ (Vite)
- **Styling**: Tailwind CSS & Framer Motion (Animations)
- **Backend/Database**: Firebase (Firestore & Auth)
- **AI Integration**: Google Gemini API
- **Icons**: Lucide React

## 🚀 Getting Started

1. **Clone & Install**:
   ```bash
   git clone https://github.com/dparamet/Bin-Scan-Better.git
   cd Bin-Scan-Better
   npm install
   ```
2. **Configure Firebase**:
   Ensure you have your `firebase-applet-config.json` in the root directory.
3. **Run Locally**:
   ```bash
   npm run dev
   ```
